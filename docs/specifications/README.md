# 仕様書ディレクトリ

このディレクトリには、システムの仕様書が格納されています。

## ドキュメント一覧

- **requirements.md**: 要件定義書 - システムの機能要件、非機能要件、制約事項、用語定義を記載
- **basic-design.md**: 基本設計書 - システム構成、データベース設計、認証・認可設計、モジュール設計、通信設計などを記載
- **screen-specification.md**: 画面仕様書 - 各画面のレイアウト、機能、認証・認可、遷移などを記載

## 関連ドキュメント

開発標準については、[`../standards/development-standards.md`](../standards/development-standards.md) を参照してください。
