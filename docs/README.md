# ドキュメントディレクトリ

このディレクトリには、プロジェクトに関する各種ドキュメントが格納されています。

## 重要なドキュメント

### 仕様書

システムの仕様に関するドキュメントは [`specifications/`](./specifications/) ディレクトリに格納されています。

- [要件定義書](./specifications/requirements.md) - システムの機能要件、非機能要件、制約事項、用語定義
- [基本設計書](./specifications/basic-design.md) - システム構成、データベース設計、認証・認可設計、モジュール設計、通信設計
- [画面仕様書](./specifications/screen-specification.md) - 各画面のレイアウト、機能、認証・認可、遷移

### 開発標準

プロジェクトの開発標準は [`standards/`](./standards/) ディレクトリに格納されています。

- [開発標準](./standards/development-standards.md) - プロジェクト全体における開発標準。コードの一貫性、可読性、保守性を向上させることを目的としています

## その他のドキュメント

技術的な詳細や分析、リファクタリング計画などのドキュメントもこのディレクトリに格納されています。
